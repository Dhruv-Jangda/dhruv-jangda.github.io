Predict Attrition

Prediction of Employee Attrition

  • Dhruv Jangda
  • Attrition Prediction

A simple deep learning project pipelined as:

  • Data loading and preprocessing done in Dask.
  • Model preparation and training done in Mxnet.
  • Training metric and model layer parameters visualization done in Tensorboard via Mxboard.

Properties

  • Scalable due to:
    • Hyperparameter and other configurations independent from project framework.
    • All pipelines i.e. loading data, pre-processing data and model training independent of each other.
  • Ease of tuning - Most of the hyperparameters are provided in the config.yaml file, thereby making model tuning easy.
  • Multiple modes - Supports both Imperative and Symbolic mode of Mxnet modelling and training.

Using Project

  • Setup python 3.6 environment with the following packages:
    • Mxnet - The project supports CPU version as only.
    • Dask
    • Dask-Dataframe
    • Mxboard
    • Tensorboard
  • Clone the repository.
  • Double click setup.py to make supporting directories.
  • Double click attrition.py to run the project.

Future Releases

  • Support for Mxnet GPU version.
  • Support for Distributed Deep Learning. Though the data is handled in a distrbuted fashion as supported by Dask, but due to symbolic or imperative mode of training, the data has to be accumulated in memory for further processing.
